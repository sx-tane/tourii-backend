passport-metadata

Get Digital Passport Metadata

Retrieve dynamic metadata for a Digital Passport NFT by token ID. This endpoint is called by NFT marketplaces and wallets to display passport information.

path Parameters
tokenId
required
string
Example: 123

The token ID of the Digital Passport NFT

Responses

/api/passport/metadata/{tokenId}
